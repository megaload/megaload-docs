Math

random

The math-random action generates a random integer between lower and upper limits.

Name Type Required Action Info
upper integer Yes No Upper limit.
lower integer Yes No Lower limit.

Example

{"math-random" : {
                "upper" : 10,
                "lower" : 2
              }
}

sum

The math-sum action sums all the integers given as arguments.

Name Type Required Action Info
integers ne-list of integers Yes Yes List of integers or actions, which must evaluate to an integer.

Example

{"math-sum" : {
                "integers" : [10, {"math-random" : {
                                                    "upper" : 10,
                                                    "lower" : 2
                                                   }
                                  }
                             ]
              }
}
